---
title: "Inspect Element of macOS system App"
description: "I bet you know serveral native Apps on macOS is written in HTML."
---

Use these commands:

```shell
defaults write NSGlobalDomain WebKitDeveloperExtras -bool true
defaults write -g WebKitDeveloperExtras -bool YES
```

<Step step="1">Open WebKitDeveloperExtras</Step>

Open your terminal and execute the two shell commands above.

<Step step="2">Check element of macOS "native" Apps</Step>

Right click on a "native" App and select "Inspect Element".
